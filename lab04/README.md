### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;

Была реализована программа на языке C# влключающее: Мультипликативный 

Отсутствие корреляции
Апериодичность
Удовлетворение равномернораспределенной С.В на [0, 1)
Воспроизводимость

Теоретическое значение:
    Выборочное среднее = 0,5
    Дисперсия = 1/12

| ----------- | MKG | Фибоначи | Встроенный random |
| ----------- | ----------- | ----------- | ----------- |
| Выборочное среднее | 0,500479 (отклонение на: 0,000479) | 0,499748 (отклонение на: -0,000252) | 0,499525 (отклонение на: -0,000475) |
| Дисперсия | 0,083712 (отклонение на: 0,000479) | 0,083196 (отклонение на: -0,000137) | 0,083383 (отклонение на: 0,000050) |

- размер выборки — **100 000 значений**;
**Вывод: ** Значение Базового датчика несущественно отличаются от значений теоретических, что делает его достоверным.
