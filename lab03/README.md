### Клеточные автоматы. Лесные пожары (GUI)

**Задание:**  
Реализовать моделирование возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата.

***1. Порывы ветра***
   Был реализован механизм с помощью которого можно промоделировать распространение огня в зависимости от ветра:
   ```
          double probability = (Math.Abs(0.05 * (double)Humidity - 0.5));
            if (Wind == "Север" && nj == 1) 
              probability *= windFactor;
            else if (Wind == "Юг" && nj == -1) 
              probability *= windFactor;
            else if (Wind == "Запад" && ni == 1)
              probability *= windFactor;
            else if (Wind == "Восток" && ni == -1)
              probability *= windFactor;
   ```
***2. Влажность***
  В регионе сущетсвует присутствие влаги в атмосфере, которая влияет на то, как долго будет гореть огонь.
  ```
    maxBurnTime = 4 - (int)(Humidity * (1/4)) + timeDay;
    maxRockStable = 30;
    maxTreeStable = 2 + (int)(Humidity * (1/4)) - timeDay;
  ```
И с какой вероятностью полыхнет дерево.
  ```
  // Если клетка пустая, то с некой вер-стью в ней появится огонь 
  if (rnd.NextDouble() < (Math.Abs(0.05 * (double)Humidity - 0.5))/2)
  {
      data[i, j] = 2;
      Health[i, j] = maxBurnTime;
  }
  // Если клетка пустая, то с некой вер-стью в ней появится дерево
  else if (rnd.NextDouble() < (Math.Abs(0.05 * (double)Humidity + 0.5)))
  {
      data[i, j] = 1;
      Health[i, j] = maxTreeStable;
  }
  ```
4. Время суток
   На распространение огня также влияет, какое сейчас время суток, поэтому при пересчете количества поинтов у огня и дерево это играет существенную роль.
5. Вода и камни
   Также были реализованы естественные преграды в виде воды и камней. Вода является постоянным блоком, поэтому огонь через неё не распространяется, а камень, напротив, имеет определённое количество очков прочности, прежде чем сгорит.
